#me-analyzer
Displays how many times, the words I, me, my, mine appear in a text file. Good for newbie bloggers 

##Usage
    
    python me.py ~/Desktop/blog.txt

##Sample Output

    Analyzing ~/Desktop/blog.txt
    Found 1134 words
             I => 6 times
            me => 0 times
            my => 4 times
          mine => 0 times
           I'm => 1 times
           you => 2 times
        you're => 4 times
          your => 5 times
